## Klávesové zkratky

| Zkratka | Co dělá |
| ----------- | ----------------- |
| `Ctrl + A` | Přepočítá a aplikuje STF |
| `F12` | Zapíná a vypíná aktuální STF |
| `Ctrl + K` | Zapíná a vypíná náhled masky |
| `Ctrl + Shift + I` | Invertuje aktivní masku |
| `Ctrl + Z` | Krok zpět v aktuálním zvoleném okně |
| `Ctrl + Y` | Krok kupředu v aktuálním zvoleném okně |
| `Alt + N` | Vytvořit nový náhled |
| `Ctrl + Shift + Z` | Krok zpět / kupředu v náhledu |
| `Space + LeftClick` | Pohyb fotkou pomocí myši |
| `Ctrl + Page Down` | Přepínání pohledu mezi dvěma posledními zvolenými fotkami |
| `Ctrl + End` | Přepínání pohledu mezi všemi fotkami |
